Chariot is a work-in-progress hackable modular desktop assistant. Think Cortana, but entirely open and customizable. The entire application is scripts running either in a GTK interface or command line,
with a live evaluation window that lets the user run any arbitrary code in real-time,
alongside a flexible extension system that gives access to the entire application runtime, letting end users
add new skills, commands, UI elements, anything they like, either by running more advanced code, or using a
convenient API and simple to understand methods for people with little programming knowledge.

Most of the application is still in early stages, and not all of the ideas are fully realised yet, but the core
is fully functional.
